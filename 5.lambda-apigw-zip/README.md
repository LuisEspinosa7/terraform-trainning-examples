# lambda-apigw-zip

In this example I build 1 lambda integrated with api gateway packaged in a zip.

## Installation

### Step 1
1. Init and download provider
```
$ terraform init
```

### Step 2
2. Provision the resources
```
$ terraform apply
```

### Step 3
3. Destroy
```
$ terraform destroy
```
