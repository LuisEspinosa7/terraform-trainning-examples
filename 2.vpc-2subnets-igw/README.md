# vpc-2subnets-igw

In this example I build 1 VPC with 2 public subnets (1 EC2 instance on each subnet) and
finally and Internet Gateway for allowing internet access.


## Installation

### Step 1
1. Init and download provider
```
$ terraform init
```

### Step 2
2. Provision the resources
```
$ terraform apply
```

### Step 3
3. Destroy
```
$ terraform destroy
```
