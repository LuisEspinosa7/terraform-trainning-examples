# dynamo-rds-s3

In this example I build 1 dynamo DB, 1 MySQL RDS DB and 1 s3 bucket.

## Installation

### Step 1
1. Init and download provider
```
$ terraform init
```

### Step 2
2. Provision the resources
```
$ terraform apply
```

### Step 3
3. Destroy
```
$ terraform destroy
```
